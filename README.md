
# 📄 AI-Powered Resume Analyzer

An intelligent Applicant Tracking System (ATS) and resume analyzer built with **Python**, **Streamlit**, and the **Google Gemini API**. This tool uses advanced Natural Language Processing (NLP) to evaluate resumes against job descriptions, calculate a match percentage, identify missing keywords, and provide actionable improvements.

## 🌟 Features
* **PDF Parsing**: Automatically extracts text from uploaded PDF resumes.
* **Semantic Role Matching**: Calculates an ATS match score based on context, going beyond simple keyword matching.
* **Skill Gap Analysis**: Identifies critical missing keywords and skills required by the job description.
* **AI Career Coach**: Generates specific, actionable feedback to optimize the resume.

## 🛠️ Tech Stack
* **Frontend UI**: [Streamlit](https://streamlit.io/)
* **NLP & AI Engine**: Google Gemini API (`google-genai` SDK)
* **Document Processing**: `PyPDF2`
